\doxysubsection{Detailed Description}


\href{https://www.sparkfun.com/products/14348}{\texttt{ {\itshape Spark\+Fun CCS811/\+BME280 Combo (SEN-\/14348)}}}

This library allows the user to\+:


\begin{DoxyItemize}
\item Read pressure in Pa
\item Read temperature in C
\item Read humidity in RH
\end{DoxyItemize}

It also provides the following mathematical functions based from the above.


\begin{DoxyItemize}
\item Read temperature in F
\item Read altitude in meters
\item Read altitude in feet
\end{DoxyItemize}

Some examples include a circular buffer class for averaging.
